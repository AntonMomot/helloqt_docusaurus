# Glossary

## C

### Contact Form 7 (CF7) {#cf7}

**_Contact Form 7_** or **_CF7_** is a WordPres plugin for creating and managing contact forms.

-   Plugin page: https://wordpress.org/plugins/contact-form-7/
-   Documentation: https://contactform7.com/docs/

### Contact Form 7 Extension For Mailchimp (ChimpMatic) {#chimpmatic}

**_Contact Form 7 Extension For Mailchimp_** or **_ChimpMatic_** is a WordPres extension plugin for
[CF7](#cf7). It adds MailChimp integration to contact forms.

-   Documentation: https://chimpmatic.com/help

## M

### MailChimp

Mailchimp is a marketing automation and email marketing platform. We use it to build
and manage mailing lists which our users can subscribe to.

-   Website: https://mailchimp.com/
